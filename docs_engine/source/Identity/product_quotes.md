# MapSnap Product Quotes

> Part of **Product Identity** — see `identity_overview.md`.  
> Voice and tone: `voice.md`. Writing rules: `writing_rules.md`.

This is NOT marketing copy.

This is NOT UI text.

This is the official language and voice of the MapSnap brand.

Every quote in this document represents a permanent design principle, product philosophy, or brand identity.

Quotes may be reused in:

- onboarding
- empty states
- README
- website
- app store
- splash screens
- presentations
- documentation

New quotes must never contradict existing ones.

Quotes should be:

- timeless
- minimal
- memorable
- authentic
- grounded in real-world use

Avoid marketing language.

Avoid exaggeration.

Avoid feature descriptions.

---

## Core Identity

"MapSnap is not a map app.
It is the fastest way to save a place that matters and return to it when it matters."

"The app captures the place before it is forgotten."

"The world is the experience.
MapSnap only remembers it."

"Capture first.
Organize later."

"Open.
SNAP.
Done."

"See something.
SNAP.
Continue living."

"Coordinates are truth."

"Your data is yours."

"Reality first.
Interface second."

"The best interface is the one you forget exists."

"The app should never become more interesting than the place being captured."

"Don't collect places.
Collect discoveries."

"Every Snap is a promise to your future self."

"The best discoveries are often the unexpected ones."

"You don't need to remember everything.
Only what matters."

"One second now.
A memory for years."

"The fastest way to return is to remember."

"The journey continues after the Snap."

"Capture the moment.
Revisit the discovery."

## Usage Rules

Quotes should never be overused.

Normally only one quote should appear on a screen.

Quotes should support the experience,
never interrupt it.

When in doubt:

remove the quote.
