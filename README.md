# MapSnap

Instantly capture places before they are forgotten.

**Capture first. Organize later.**

## Run

```bash
npm install
npm run dev
```

Open [https://localhost:3000](https://localhost:3000).

For mobile testing over Tailscale/LAN, use `npm run dev:network` and open
`https://<your-machine-ip>:3000`. See `docs_engine/source/local_https_development.md`.

## Usage

- **Tap SNAP** — save current GPS location
- **Long press SNAP** — attach a photo and save location

## Stack

Next.js · TypeScript · Tailwind · Geolocation API · IndexedDB · PWA
